Stock Performance Dashboard

Author: Xinyu Zhang
Product: Stock Performance Analysis Dashboard
Output: Interactive Tableau Dashboard

Background and Overview

This project analyzes stock market performance using historical price and trading volume data. The goal is to transform raw financial time-series data into clear visual insights that help evaluate price trends, trading activity, and investment performance.
The dashboard integrates candlestick charts, moving averages, volume indicators, and indexed price analysis to help users understand how stock prices evolve over time and how trading activity supports price movements.The analysis focuses on identifying price trends, evaluating market momentum, and providing a simple framework to assess how an investment would perform over a selected time period.
